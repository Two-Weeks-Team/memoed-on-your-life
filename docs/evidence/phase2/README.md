# Phase 2 physical-device evidence

These screenshots were attached by XCUITest while running the signed development build on an iPhone 13 with iOS 26.5.

- `iphone13-recording.png` shows the real microphone recording state.
- `iphone13-timed-transcript.png` shows the finalized on-device Speech transcript and its exact time range.

The spoken sentence is synthetic test data. No personal photo was selected or published.
